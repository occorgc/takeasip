# Questo file è un segnaposto per l'icona dell'app in formato ICNS
# Per creare un vero file ICNS, usa il seguente comando:
# 
# iconutil -c icns Resources/AppIcon.iconset 
# 
# Dove AppIcon.iconset è una cartella contenente le icone nei vari formati:
# - icon_16x16.png
# - icon_16x16@2x.png
# - icon_32x32.png
# - icon_32x32@2x.png
# - icon_128x128.png
# - icon_128x128@2x.png
# - icon_256x256.png
# - icon_256x256@2x.png
# - icon_512x512.png
# - icon_512x512@2x.png
